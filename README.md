# Enterprise-Grade Network Lab

This project documents my hands-on labs as I prepare for the CCNA by designing and building a multi-phase enterprise-style network lab.

These labs are structured to simulate a real-world network environment, with each phase introducing a new layer of security, connectivity, and functionality. 

---

## Project Goals

- Apply networking concepts in a practical lab environment  
- Simulate enterprise network design and growth  
- Implement security best practices from the ground up  
- Build and document configurations in a structured, scalable way  

---

## Lab Structure

The project is organized into phases, with each part building on the previous one.

### Part 1: Secure Device Management
Focus:
- Establish secure access to network devices  
- Validate basic network connectivity  
- Prepare the network for future expansion  

Includes:
- Console and privileged EXEC protection  
- SSH-only remote access  
- Management VLAN implementation  
- End-to-end connectivity testing (ICMP)  

See details: `part-1-secure-management/`

---

## Upcoming Phases

- **Part 2: Access Control & Segmentation**
  - Restrict management access using ACLs  
  - Implement port security  
  - Control traffic between network segments  

- **Part 3: Advanced Network Services** *(planned)*
  - VLAN expansion and inter-VLAN routing

- **Part 4: Routing & Scalability** *(planned)*
  - Dynamic routing protocols (OSPF/EIGRP)
